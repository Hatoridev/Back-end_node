Bem-vindo à Aula 04!

Vamos explorar o uso do nó **Switch** para filtrar mensagens com base no conteúdo de `payload`.

Objetivo da Aula

Na aula de hoje, aprenderemos a:

  • Manipular mensagens utilizando o nó **Function**.
  • Filtrar mensagens com o nó **Switch** baseado no conteúdo de `payload`.

Fluxo do Node-RED

O fluxo da aula consiste nos seguintes passos:

  • **Inject**: Envia uma mensagem inicial.
  • **Function**: Modifica a mensagem adicionando um texto extra.
  • **Switch**: Filtra a mensagem, verificando se o texto contém a palavra "Fluxo".
  • **Debug**: Exibe a mensagem final no painel de debug.

Passos para rodar no Termux

  1. Clone o repositório.
  2. Instale o Node.js e o Node-RED.
  3. Execute o Node-RED: node-red.
  4. Importe o fluxo do arquivo flows.json no Node-RED.
  5. Acesse o Node-RED em localhost:1880.
