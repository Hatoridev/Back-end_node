Bem-vindo à Aula 05!

Vamos criar uma API simples em Node-RED que retorna dados simulados sobre um usuário.

Objetivo da Aula

Na aula de hoje, aprenderemos a:

  • Criar uma API no Node-RED com o nó **HTTP In**.
  • Retornar dados simulados usando o nó **Function**.
  • Responder a requisições HTTP com o nó **HTTP Response**.

Fluxo do Node-RED

O fluxo consiste nos seguintes passos:

  • **Inject**: Envia uma mensagem inicial com dados simulados.
  • **HTTP In**: Define uma rota de API para receber requisições GET.
  • **Function**: Manipula a mensagem e define a resposta da API com dados simulados.
  • **HTTP Response**: Envia a resposta da API para o cliente.

Passos para rodar no Termux

  1. Clone o repositório.
  2. Instale o Node.js e o Node-RED:
    3. pkg install nodejs
npm install -g --unsafe-perm node-red
  3. Execute o Node-RED:
    4. node-red
  4. Implemente o fluxo com o arquivo flows.json.
  5. Acesse a URL [http://localhost:1880](http://localhost:1880) no navegador.
  6. Acesse a API em http://localhost:1880/api/usuario para ver os dados simulados.
